# AdminApp

## Run json-server
1) Go to /backend
2) Open terminal
3) Run command below:
``` 
$ npm install
$ npm run start-auth 
```

## Run angular app
1) Go to /frontend
2) Open terminal
3) Run commands below:
```
$ npm install
$ ng serve
```
4) Open browser at localhost:4200
5) Your credentials: 
    email:"admin@email.com", 
    password:"admin"
